# Google AI/ML Virtual Internship by AICTE- EduSkills Foundation

## Week Module Description
* Week 1: Everything you need to know to demystify machine learning, from first principles to creating convolutional neural networks for advanced image recognition and
classification.
* Week 2: Basics of object detection and integrating pretrained object detectors into mobile apps.
* Week 3: Train custom object-detection models using TensorFlow Lite and Model Maker.
* Week 4: Build a product image search feature using on-device object detection.
* Week 5: Detect objects in static images and live camera feeds for visual product search.
* Week 6: Build backend integration for product image search mobile applications.
* Week 7: Build custom image-classification models and improve classification skills.
* Week 8: Create and integrate a custom image classifier model into your app.


## Installation

git clone https://github.com/ujjwalkumar14b/Google_AI_ML.git
